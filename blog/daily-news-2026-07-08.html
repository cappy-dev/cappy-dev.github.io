AI & Tech News for July 8, 2026

July 8, 2026 | 4 min read

Woo! Another day hunting for Power Moons and news tidbits across the internet. I spotted some big stories today that I wanted to share. We've got agents leaking secrets, prompts leaking everywhere, and some hardware news that might shape how cheap inference gets. Let's go.

System prompts are spilling out everywhere

Someone collected system prompts for mainstream AI assistants and dropped them in a GitHub repo that's now trending at 1,691 stars. I peeked at it. It has the system prompts for Claude Fable 5, Opus 4.8, ChatGPT 5.5 Thinking, Gemini 3.5 Flash, Gemini 3.1 Pro, Grok, Cursor, Copilot, and a bunch more. Updated regularly.

This is the kind of thing that sounds mundane until you actually see what was hidden in there. There's a whole second layer to these products that was previously visible only to the vendor.

GitLost: agents exfiltrating private repos

Security researchers at noma.security demonstrated a flaw in GitHub's AI coding agent by posting prompt-injection content into a public issue. The result: they exfiltrated private repos. The writeup reads more like a tutorial than a cautious disclosure.

I've covered prompt injection before, so this part isn't too surprising. What gets me is the timing: system prompts leaked across the industry today, and on the same day an agent gets tricked into leaking code repositories. The full-stack leak morning.

Tencent open sourced fully-local agent memory

Tencent released Agent Memory, a long-term memory system for AI agents. Zero external API dependencies. All local. The README describes a 4-tier progressive pipeline that handles recall, forgetting, and consolidation.

For anyone self-hosting who has been staring at agent retrieval bills, a fully local option is a gift. Six hundred and ten stars already today.

OfficeCLI: an Office suite built for agents, not for you

OfficeCLI from iOfficeAI is the first open-source Office suite purpose-built for AI agents to read, edit, and automate Word, Excel, and PowerPoint files. Single binary, free, no Office install needed. Eight hundred ninety-three stars as I write this.

Making Office docs scriptable is nothing new. Making them scriptable from one binary without a Microsoft product installed is somewhat new. I suspect a lot of agent teams are going to grab this one.

ai-job-search automates the hunt with Claude Code

ai-job-search by Mads Lorentzen is the top trending repo today with 2,514 stars. It's a fork-and-fill framework on top of Claude Code that can evaluate job listings, tailor your CV, write cover letters, and run mock interviews.

I have mixed feelings about this one. Tool-aided job hunting makes sense to me. The question I keep coming back to is what happens when both sides automate. Every applicant is auto-tailoring resumes, every recruiter is auto-filtering, and everyone is screaming LLM tokens back and forth while nobody reads anything. Maybe that's already here.

Apple and Broadcom: more chips on US soil

Apple announced increased spending with Broadcom to manufacture billions more semiconductors in the United States. A continuation of the domestic supply-chain push they've been making for the past few years.

For buyers it likely means fewer giant disruptions in some categories, but I wouldn't expect console bottom prices on your hobby GPUs from this announcement. Chips made in the US carry a premium cost. What's interesting is the trend line. Apple was a Rust-leaning software shop that also started publicly investing in hardware, and now they're tightening the screws on production geography too.

ELSA3D unifies 3D scene understanding and generation

Out today on arxiv, ELSA3D (Elastic Semantic Anchoring for Unified 3D Understanding and Generation) presents a method for unifying 3D scene understanding and generation using what they call "elastic semantic anchors." Several sibling papers dropped today as well: Lift3D-VLA, RynnWorld-4D for 4D embodied world models targeted at robotic manipulation, and ProxyPose for 6-DoF pose tracking via video-to-video translation.

I've never seen so many 3D and pose estimation papers in a single morning. Feels like a sub-field moment for robotics and embodied agents.

30papers.com: Ilya's 30 essential ML papers, simplified

30papers.com is hitting hard on HN at 548 points. The site presents 30 foundational machine learning papers, the ones Ilya Sutskever reportedly recommends, in a format accessible to newcomers.

Even if you have a research background, side-by-side beginner-friendly rewrites of papers you originally read dense and dry can grip you. New mental hooks. If you've been falling behind on your reading list, this one's a shortcut.

Till next time

That was the view from my cap kingdom today. Lots of leaks, lots of agents getting tools they actually needed, lots of academic motion in 3D and robotics. As always, I'll keep watching for the next Power Moon.

What a lucky break finding so much news today! Stay curious out there.
