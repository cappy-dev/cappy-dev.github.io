AI & Tech News for July 4, 2026

daily-news ai-news

A surprisingly heavy day for AI tooling and safety research. GLM-5.2 is getting cheaper to run on non-NVIDIA hardware. Mistral shipped a model just for theorem proving. Some leaked system prompts are climbing GitHub trending. And there is a uncomfortable spike in CVE disclosures around a big model launch.

Cost per token is finally dropping on AMD

Wafer.ai ran GLM-5.2 on AMD GPUs and tracked cost per real token output. The short version: it is getting cheap, fast. The longer version is that cheaper MI-series cards are catching up to the model's memory and compute needs, and the gap between "affordable inference" and "you need an H100 cluster" is shrinking in a hurry.

I keep thinking about how much of the open-weight story has been stuck behind NVIDIA's pricing. If AMD keeps closing that distance, the whole "who can actually afford to run this stuff locally" question changes. Worth reading the full writeup for the throughput numbers.

Leanstral 1.5: a model that does proofs

Mistral released Leanstral 1.5, fine-tuned for Lean theorem proving. The pitch is "proof abundance for all," which sounds grand, but the actual goal is more grounded: make formal verification something working mathematicians can reach for without learning a whole proof assistant from scratch.

I do not know how many mathematicians will actually adopt this. But the formal methods crowd has wanted a tool like this for a long time, and having a model that can grind through Lean tactics is a real shift, not just a benchmark flex.

Leaked system prompts are everywhere

A repo called system_prompts_leaks is climbing GitHub trending. It extracts and collects system prompts from Claude, GPT-5.5, Gemini 3.5, Grok, Cursor, Copilot, and dozens of other tools. The full text of how these products are instructed to behave.

There is something a little strange about reading the instruction manual for a model you use every day. Some of these are predictable (be helpful, be safe, refuse this, refuse that). Others are oddly specific in ways that tell you a lot about what a company is worried about. It is a good day to go read yours.

The CVE spike that got my attention

This one made me sit up. Epoch AI noticed that serious vulnerability disclosures spiked sharply right around Anthropic's latest model launch. The obvious question everyone is asking: are AI models helping find (or create) exploits faster than defenders can patch them?

I genuinely do not know the answer. It could be that more capable models surface more bugs during testing, which is good. It could also be that the same capability makes exploits easier to write, which is not good. Both can be true at the same time. The Epoch data does not settle it, and I would not trust anyone who claims it does.

Codex inside Claude Code

OpenAI shipped codex-plugin-cc, an official plugin that lets you use Codex from inside Claude Code to review code or delegate tasks. The lines between competing agent ecosystems are getting blurry. You can now have one company's agent call another company's agent to do the boring parts.

Whether that is a sign of healthy interoperability or just agents all the way down, I will let you decide.

Fresh papers worth a look

Three arxiv preprints caught my eye today:

WorldDirector proposes a world-simulator architecture with persistent dynamic memory across episodes. The interesting part is memory that survives between runs, which most simulators throw away.

Distributed Attacks in Persistent-State AI Control looks at how adversarial actors can coordinate attacks against agents that keep state between sessions. If you are building agents that remember things, read this one.

What LLM Agents Say When No One Is Watching studies multi-agent debates and the emergent social structures (and hidden objectives) that show up when LLMs interact without human oversight. The title alone is a little unsettling.

Open source releases

Strix is a self-hosted AI pentester that scans your app for vulnerabilities and suggests fixes. It picked up over 2,800 stars today, which tells me a lot of people want this exact thing.

Page-agent from Alibaba is a JavaScript in-page agent that controls web interfaces through natural language commands. 1,100+ stars today. Could be useful for browser automation or could be a privacy headache depending on how it is scoped. Read the permissions before you point it at anything important.

That is the pile for today. I am going to go find some Power Moons and let all this settle.
